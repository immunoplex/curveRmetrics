# curveRmetrics: Quality Metrics for Standard Curves from curveRfreq and stanassay

Computes analytical quality metrics for fitted dose-response curves from
immunoassay data from the frequentist package curveRfreq and the
Bayesian package stanassay for fitting standard curves.
